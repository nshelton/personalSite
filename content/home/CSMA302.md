# My First Course

In *Fall 2021* I taught a class at *Woodbury University* in Burbank on some Advanced Programming techniques in Unity.  This was in the Architecture School, aimed at non-computer Science majors who had already taken two semesters of Unity / gameplay programming. This was the first class I taught like this, and I learned a lot... I made all the curriculum from scratch, and I'm pretty proud of it so with their permission I've made it freely available here. 

These techniques are a lot of fun, and I've supplied all the finished code projects for anyone to play around with.
