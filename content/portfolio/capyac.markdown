# VJ for CAPYAC


<iframe width="100%" height="450" scrolling="no" frameborder="no" src="https://w.soundcloud.com/player/?url=https%3A//api.soundcloud.com/users/41257&amp;auto_play=false&amp;hide_related=false&amp;show_comments=true&amp;show_user=true&amp;show_reposts=false&amp;visual=true"></iframe>



---

in 2015 I was lucky enough to meet the talented folks at CAPYAC and become a part of their live show. I use many types of software like VDMX, WebGL and Magic Music Visuals to run custom GLSL shaders and ISF. By hooking a direct audio feed from the stage mixer into the shaders I can get some really responsive audio reactive stuff going on.

Source for most of these can be found at interactiveshaderformat.com [ ISF Source](http://www.interactiveshaderformat.com/u/sheltron3030)

To open in VDMX, copy the source to a .fs file and drag the file into the Media Bin.
